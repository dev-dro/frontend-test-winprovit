The Alticci Sequence

The Alticci sequence - a(n) is defined by the formulas:

  • n = 0 => a(0) = 0
  • n = 1 => a(1) = 1
  • n = 2 => a(2) = 1
  • n > 2 => a(n-3) + a(n-2)
